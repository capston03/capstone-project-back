## 디렉토리 구성

- blueprints: blueprint 객체
- data: Test용 2D 이미지/3D 모델
- handler: DB, Storage 접근 핸들러
- image_processing: 
  - 2D -> 3D 변환 
  - 배경과 전경 분리
- model: DB와 연동하여 사용자가 입력한 데이터나 사용자에게 출력할 데이터
- namespaces: namespace 객체
- utility: 공용으로 이용 가능한 코드
